# "System" Challenges

System challenges are designed to test your understanding of how operating systems work, including concepts like processes, memory management, file systems, and more. These challenges often require you to interact with the underlying system, manipulate files, or analyze system behavior.

Challenges:
- [Homelab Pwnlab](./homelab-pwnlab.md): A challenge that simulates a homelab environment where you need to find and exploit vulnerabilities to gain access.
